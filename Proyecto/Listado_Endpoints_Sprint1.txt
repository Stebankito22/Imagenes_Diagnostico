=====================================================
INTERACCIONES CON OTROS MICROSERVICIOS
=====================================================

Este microservicio de Imagen Diagnostico se integra con:

1. Microservicio de Pacientes
   - Recibe: PacienteId (en OrdenesImagen y EstudiosRealizados)
   - Consulta datos del paciente mediante PacienteId

2. Microservicio de Medicos
   - Recibe: MedicoSolicitanteId (en OrdenesImagen)
   - Consulta datos del medico solicitante

3. Microservicio de Contabilidad (su microservicio)
   - Envía: Datos de estudios realizados para facturación
   - Recibe: Confirmación de pagos y estados de cuenta

=====================================================
NORMALIZACION: 5FN
=====================================================

Las tablas estan normalizadas hasta 5FN:
- Equipos: Codigo (PK), Nombre, Modalidad
- TiposEstudio: Id (PK), Codigo, Nombre, Modalidad, PrecioBase, RequiereContraste
- TecnicosEjecutores: Id (PK), Codigo, Nombre, Especialidad
- OrdenesImagen: Id (PK), Codigo, PacienteId, MedicoSolicitanteId, FechaSolicitud, Urgencia
- EstudiosRealizados: Id (PK), Codigo, OrdenImagenId (FK), TipoEstudioId (FK), 
                       TecnicoEjecutorId (FK), EquipoId (FK), FechaHoraInicio, 
                       FechaHoraFin, DosisRadiacion, ContrasteAplicado, EstadoInforme
